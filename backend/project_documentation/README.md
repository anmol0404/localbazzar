# Project Documentation

This folder contains the **definitive** technical documentation for the LocalBazaar platform. These documents are designed to be "Production-Ready" and cover every aspect of the system in detail.

## Files

### 1. [Technical Specification](./technical_specification.md)
**The Master Plan.** Contains:
-   **System Architecture Diagram**: Visual flow of data and services.
-   **Complete Database Schema**: 20+ Prisma Models with all fields, enums, and indexes.
-   **Redis Strategy**: Exact key patterns for caching and rate limiting.
-   **AI Integration**: Detailed flows for Smart Search and Negotiation Chatbots.

### 2. [Backend Architecture](./backend_structure.md)
**The Codebase Guide.** Contains:
-   **Modular Monolith Structure**: Detailed directory layout.
-   **Design Patterns**: Controller-Service-Repository explanation.
-   **Key Technologies**: How to implement BullMQ, Zod, and Error Handling.

### 3. [Frontend Architecture](./frontend_structure.md)
**The Client-Side Guide.** Contains:
-   **Web (Next.js)**: App Router structure, Server vs Client state.
-   **Mobile (Flutter)**: Clean Architecture, Offline-first strategy.
-   **Design System**: Shared colors, typography, and component guidelines.
